Estruturas condicionais

Condicional simples

Estruturas condicionais servem para açterar o fluxo de execução do meu algoritmo. Após um teste de condição, se a condição for verdadeira, o fluxo de exucução interno do código interno à estrutural condição executa. Caso contrário, o código salta para o final da estrutura condicional

if(condicao == verdadeira){ comandos executados quando a condição for verdadeira } else{ comandos executados quando a condição é falsa }
